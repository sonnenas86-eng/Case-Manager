
File Movement and Case Management System - Starter Version

How to Run:
1. Open VS Code.
2. Open the folder containing this project.
3. Open a terminal in VS Code.
4. Run: pip install flask
5. Run: python app.py
6. Open your browser and go to http://127.0.0.1:5000/

Features:
- Dashboard overview of cases
- Register new cases
- Track file movements

Enjoy!
